## 2. 哈夫曼树的构造

```
HuffmanTree Huffman(MinHeap H)
{
    HuffmanTree T;
    BuildMinHeap(H);
    for(int i=1;i<H->Size;i++)
    {
        T=malloc(sizeof(struct TreeNode));
        T->Left=DeleteMin(H);
        T->Right=DeleteMin(H);
        T->Weight=T->Left->Weight+T->Right->Weight;
        Insert(H,T);
    }
    T=DeleteMin(H);
    return T;
}
```
