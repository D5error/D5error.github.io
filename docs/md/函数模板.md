## 1. 函数模板

```
template<class 类型参数1,class 类型参数2,...>
返回值类型 模板名(形参表)
{
    函数体;
}
```

```
template <class T>
void Swap(T &x,T &y)
{
    T tmp=x;
    x=y;
    y=tmp;
}
int main()
{
    int n=1,m=2;
    Swap(n,m);    //编译器自动生成void Swap(int &,int &)函数
    double f=1.2,g=2.3;
    Swap(f,g);    //编译器自动生成void Swap(double &,double &)函数
    return 0;
}
```

* 函数模板中可以不只一个类型参数

```
template <class T1,class T2>
T2 print(T1 arg1,T2 arg2)
{
    cout<<arg1<<" "<<arg2<<endl;
    return arg2;
}
```

* 不通过参数实例化函数模板

```
#include<iostream>
using namespace std;
template <class T>
T Ind(T n)
{
    return 1+n;
}
int main()
{
    cout<<Inc<double>(4)/2;    //输出2.5
    return 0;
}
```
