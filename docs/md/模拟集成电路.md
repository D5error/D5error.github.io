# 直流偏置电路

![Alt text](image-581.png)

* 模拟集成电路中普遍采用**直流源**做直流偏置

* 为各级放大电路提供偏流以**稳定静态工作点**

* 作为**有源负载**取代不易集成的大电阻

![Alt text](image-493.png)

![Alt text](image-492.png)

* 晶体管和场效应管等有源器件，在放大区其输出电流几乎不受输出电压的影响，具有恒流特性，因而可以用来设计各种性能良好的**恒流源**
