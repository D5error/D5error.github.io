## 1. new运算符

> ### *分配一个变量*
 
`P=new T;`
* T是任意类型名,P是类型为T*的指针
* 动态分配出一片大小为sizeof(T)字节的内存空间,并将该内存空间的起始地址赋值给P

> ### *分配一个数组*

`P=new T[N];`
* T是任意类型名,P是类型为T*的指针,N是要分配的数组元素的个数,可为表达式
* 动态分配出一片大小为sizeof(T)*N字节的内存空间,并将该内存空间的起始地点赋值给P

## 2. delete运算符

* 用`new`动态分配的内存空间,一定要用`delete`运算符进行释放

> ### *释放一个变量*

```
int *P=new int;
*P=5;
delete P;
```

> ### *释放一个数组*

```
int *P=new int[20];
P[0]=1;
delete[] P;
```
