## 1. 希尔排序

* 原始希尔排序

```
void Shell_sort(ElementType A[],int N)
{
    for(int D=N/2;D>0;D/=2)
    {
        for(int P=D;P<N;P++)
        {
            Tmp=A[P];
            for(int i=P;i>=D&&A[i-D]>Tmp;i-=D)
            {
                A[i]=A[i-D];
            }
            A[i]=Tmp;
        }
    }
}
```
