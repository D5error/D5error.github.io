# 贪心算法
* 一般步骤：
    1. 将问题分解为若干子问题
    2. 找出适合的贪心策略
    3. 求解每一个子问题的最优解
    4. 将局部最优解堆叠成全局最优解


* 1.分发饼干

    ![Alt text](image-371.png)

    思路1：将剩下最小尺寸的饼干给剩下最小胃口且满足胃口的孩子，保证饼干最小程度没有浪费

    思路2：将剩下最大尺寸的饼干给剩下最大胃口且满足胃口的孩子，保证饼干能喂饱尽可能多的孩子
