# 图
* 边的定义

```cpp
typedef int VertexType;    //顶点
typedef int WeightType;    //边的权值
typedef struct ENode *PtrToENode;
struct ENode
{
    VertexType V1,V2;    //有向边<V1,V2>
    WeightType Weight;    //边的权重
}
typedef PtrToENode Edge;
``` 

* 图顶点的定义

```cpp
typedef struct GNode *PtrToGNode;
struct GNode
{
    int NumVertex;    //顶点总数
    int NumEdge;    //边总数
    WeightType G[MaxVertexNum][MaxVertexNum];    //邻接矩阵
    DataType Data[MaxVertexNum];    //存顶点的数据
};
typedef PtrToGNode MGraph;
```

* 图的初始化（无边，初始化顶点）

```cpp
MGraph CreateGraph(int VertexNum)
{
    Vertex V,W;    //顶点坐标
    MGraph Graph=new GNode;
    Graph->NumVertex=VertexNum;
    Graph->NumEdge=0;
    for(V=0;V<Graph->NumVertex;V++)//初始化邻接矩阵
    {
        for(W=0;W<Graph->NumVertex;W++)
        {
            Graph->G[V][W]=INFINITY;
        }
    }
    return Graph;
}
```

* 插入边

```cpp
MGraph BuildGraph(MGraph Graph,Edge E)
{
    Graph->G[E->V1][E->V2]=E->Weight;
    Graph->G[E->V2][E->V1]=E->Weight;
}
```
