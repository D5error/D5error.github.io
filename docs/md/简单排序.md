## 2. 插入排序

```
void Insertion_Sort(ElementType A[],int N)
{
    for(int p=1;p<N;p++)
    {
        Tmp=A[p];
        for(int i=p;i>0&&A[i-1]>Tmp;i--)
        {
            A[i]=A[i-1];
        }
        A[i]=Tmp;
    }
}
```
