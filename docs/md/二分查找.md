# 二分查找
```cpp
int BinarySearch(List Tbl,ElementType K)
{
    int left,right,mid,NoFound=-1;
    left=1;//从1或从0，取决于序列开头下标，下面同理
    right=Tbl->Length;
    while(left<=right)
    {
        mid=(left+right)/2;
        if(K<Tbl->Element[mid])
        {
            right=mid-1;
        }
        else if(K>Tbl->Element[mid])
        {
            left=mid+1;
        }
        else
        {
            return mid;
        }
    }
    return NotFound;//没找到时，left>right
}
```
