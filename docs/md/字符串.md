# 2. 字符串
* 用引号括起都是字符串,引号可为单引号或双引号

|操作|用途|
|-|-|
|name.title()|将每个单词的首字母改为大写,其余小写|
|name.upper()|全部改为大写|
|name.lower()|全部改为小写|
|first_name+second_name|将字符串合并|
|name.rstrip()||

# 3. 字符串运算符
|运算符|功能|示例|
|:-|:-|:-|
|+|连接字符串|>>>'D5'+'rrr'`D5rrr`|
|*|复制字符串|>>>'ab'*3`ababab`|

# 4. 字符串切片
* 字符串可是正向递增也可以是反向递减

|0|1|2|3|4|5|6|7|
|-|-|-|-|-|-|-|-|
|a|b|c|d|e|f|g|h|
|-8|-7|-6|-5|-4|-3|-2|-1|

* 索引:在[]中给出序号
* 切片在[]中给出切片序号范围 >>>[1:5]`bcde`
* [起始(包含):结束+1:步长]
